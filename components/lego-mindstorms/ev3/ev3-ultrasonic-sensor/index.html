    EV3 Ultrasonic Sensor¶

    Category Version Requires
    LEGO® MINDSTORMS® > Ev3 1 API 19, Android 4.4 - 4.4.4 KitKat

    Non-visible component

    Overview¶

    A component that provides a high-level interface to an ultrasonic sensor on a LEGO MINDSTORMS EV3 robot.

    Events¶

    Above Range¶

    Called when the detected distance has gone above the range.

    Below Range¶

    Called when the detected distance has gone below the range.

    Within Range¶

    Called when the detected distance has gone within the range.

    Methods¶

    Get Distance¶

    Block preview not available

    Returns `number`

    Returns the current distance in centimeters as a value between 0 and 254, or -1 if the distance can not be read.

    Set CM Unit¶

    Block preview not available

    Measure the distance in centimeters.

    Set Inch Unit¶

    Block preview not available

    Measure the distance in inches.

    Properties¶

    Above Range Event Enabled¶

    Available as Common Property

    Read-Write property

    Type Default
    boolean False

    Whether the AboveRange event should fire when the distance goes above the TopOfRange.

    Below Range Event Enabled¶

    Available as Common Property

    Read-Write property

    Type Default
    boolean False

    Whether the BelowRange event should fire when the distance goes below the BottomOfRange.

    Bluetooth Client¶

    Available as Common Property

    Read-Write property

    Type
    component

    The BluetoothClient component that should be used for communication.

    Bottom Of Range¶

    Available as Common Property

    Read-Write property

    Type Default
    number 30

    The bottom of the range used for the BelowRange, WithinRange, and AboveRange events.

    Sensor Port¶

    Available as Common Property

    Read-Write property

    Type Default
    text 1

    The sensor port that the sensor is connected to.

    Top Of Range¶

    Available as Common Property

    Read-Write property

    Type Default
    number 90

    The top of the range used for the BelowRange, WithinRange, and AboveRange events.

    Unit¶

    Available as Common Property

    Read-Write property

    Type Default
    text cm

    Specifies the unit of distance.

    Within Range Event Enabled¶

    Available as Common Property

    Read-Write property

    Type Default
    boolean False

    Whether the WithinRange event should fire when the distance goes between the BottomOfRange and the TopOfRange.
